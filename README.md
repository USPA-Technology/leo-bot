# leo-bot

## Checklist before start

* Need Python 3
* sudo apt install python-is-python3
* pip install virtualenv
* Need to create a file .env and add OPENAI_API_KEY 
* Set correct DIR_PATH in start_app.sh
* Run ./start_app.sh
